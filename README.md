# Krish Jain

CS at the University of Rochester, focused on systems engineering, security, and AI. These days I spend most of my time arguing with stochastic parrots.

## Work

**[Microsoft](https://microsoft.com)**: I built testing infrastructure for M365 Substrate, the exabyte-scale distributed storage layer that powers Teams, Exchange, and SharePoint.

**[Chainguard](https://chainguard.dev)**: I worked on secrets detection pipelines for container images and automated CVE triage and patching for [Wolfi](https://github.com/wolfi-dev), a distroless Linux distribution designed for minimal attack surface. This involved building tooling to scan packages for leaked credentials and integrate vulnerability data into the build system. During my time there, the company [raised its $140M Series C](https://www.chainguard.dev/unchained/chainguard-raises-140-million-in-series-c-funding-to-secure-the-next-frontier-of-ai-workloads) led by [Redpoint Ventures](https://redpoint.com), with participation from [Sequoia](https://sequoiacap.com), [Spark Capital](https://sparkcapital.com), and [Amplify](https://amplifypartners.com).

**[Flatcar Linux](https://flatcar.org)**: I replaced the deprecated Torcx addon system with systemd-sysext, enabling runtime-composable OS extensions while preserving immutability guarantees. This was with the Kinvolk team, which was later acquired by Microsoft.

**Research**: I published a paper at IEEE on ML-based security policy analysis and contributed literate executable support to GNU Screen.
